# AyatPrint Roadmap

## Completed ✅
- Disaggregate monolithic codebase into Ayat Studio, Core API, and Storefront POC.
- Establish comprehensive Open Source Wiki and Documentation.
- Dockerize all core subsystems.
- Implement GitHub Actions CI/CD pipelines.

## In Progress 🚧
- **PIM Integration:** Building the dedicated Product Information Management dashboard to securely store and translate Quranic verses without hardcoding them into the Studio.
- **Headless Commerce Cutover:** Finalizing the decision between WordPress and Magento, and migrating the production storefront.

## Short-Term (Next 3 Months) 🗓
- **Advanced Room Simulator:** Introduce augmented reality (AR) previews using WebXR.
- **Automated Localization:** Add multi-language support (Arabic, French, English) to the Ayat Studio UI.

## Medium-Term (6 - 12 Months) 🚀
- **B2B Portals:** Create dedicated login areas for Interior Designers to manage bulk projects, quotes, and custom typography scaling.
- **AI Upscaling:** Integrate open-source AI models to upscale user-uploaded artwork to 300 DPI for massive canvas prints.

## Long-Term (1+ Years) 🌌
- **Federated API:** Open the Ayat Core API to third-party developers, allowing other Muslim-focused ecommerce sites to utilize our typographic rendering engine and fulfillment network.
